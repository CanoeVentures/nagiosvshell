About

Nagios V-Shell is an alternative way to view Nagios output. It installs alongside Nagios, so there is no risk to try. It is compatible with both Nagios Core and Nagios IX.

The latest generation features:

  • Familiar interface for existing Nagios users
  • Modern and responsive design
  • Dynamicly updated frontend built in AngularJS
  • Searchable, sortable, filterable tables with FooTable
  • Quicksearch functionality
  • RESTful API powered by CodeIgniter

Quick Install

$ git clone https://github.com/chrislaskey/nagiosvshell
$ vim nagiosvshell/config.php
$ ./nagiosvshell/install.php

Then point your browser to /vshell2.

For complete installation directions see the latest read me.
